## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## storybook

```bash
yarn storybook
```

Open [http://localhost:6006](http://localhost:6006) with your browser to see the result.
